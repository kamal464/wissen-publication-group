# Superscript Characters Quick Reference

## Copy-Paste These Characters:

### Superscript Numbers:
```
¹ ² ³ ⁴ ⁵ ⁶ ⁷ ⁸ ⁹ ⁰
```

## Usage Examples:

### Author Names:
```
Bharath¹, Vinay², Ravi³
John¹, Jane², Bob³
```

### Author Affiliations:
```
¹ Department of Computer Science, ² Department of AI
¹ XYZ University, ² ABC Institute
```

## Quick Access Methods:

### Method 1: Windows Emoji Picker (Easiest)
1. Click in the input field
2. Press `Win + .` (Windows key + period)
3. Click "Symbols" tab
4. Find and click superscript numbers

### Method 2: Character Map
1. Press `Win + R`
2. Type `charmap` and press Enter
3. Search for "superscript"
4. Double-click to copy, then paste

### Method 3: Alt Codes
Hold `Alt` and type on numeric keypad:
- ¹ = Alt + 0185
- ² = Alt + 0178
- ³ = Alt + 0179

### Method 4: Copy from this file
Just copy the characters above and paste them!

## Tips:
- You can type regular text and add superscripts anywhere
- Example: `Bharath¹` or `John² Smith`
- Superscripts work in both Author Names and Affiliations fields
- Characters are stored exactly as you type them
